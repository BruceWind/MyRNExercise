# MyRNTest


## how to run it.

### At the first time.
```
npm install 
```

### run it.
Connecting an emulator or device in advance of this step.

1. Open a couple of terminals.

2. one of them run:
```
npm run start  # it will run in background. To close it, close that terminal used. 
``` 

3. another of them run:
```
npm run android
```
